# Loan Prediction App

## Overview
This project predicts the loan amount a user can take based on their financial and personal information using a trained Machine Learning model (Random Forest Regressor). The app is built using Streamlit for easy web deployment.

## Features
- Input fields for:
  - Married status
  - Education
  - Self-employment
  - Number of dependents
  - Total income
- Predict loan amount instantly
- User-friendly web interface
